# Low-Code Schulverwaltung

Dieser Ordner enthält einen einfachen Low-Code/KI-Prototyp der Schulverwaltung.
Die App ist absichtlich einfacher als die Angular-Version und dient zum Vergleichen der zwei Ansätze.

## Starten

Die Datei `index.html` kann direkt im Browser geöffnet werden.
Es ist kein Build-Schritt notwendig.

## Funktionen

- Dashboard mit Anzahl von Schülern, Lehrern und Räumen
- Schüler über Formular hinzufügen
- Schüler aus der Liste löschen
- Beispieldaten für Lehrer und Räume anzeigen
- Speicherung der Schüler im Browser über `localStorage`

## Unterschied zur Angular-App

Die Angular-App ist eine richtige Frontend-Applikation mit Komponenten, Service und HTTP-Zugriff auf das Backend.
Diese Low-Code-Version ist ein einfacher Prototyp mit HTML, CSS und JavaScript. Sie ist schneller erstellt, aber technisch weniger sauber und nicht direkt mit dem Backend verbunden.

## Lessons Learned

- Low-Code/KI ist gut für einen schnellen Prototyp.
- Für eine echte Anwendung ist Angular strukturierter.
- Die Angular-Version ist besser erweiterbar.
- Die Low-Code-Version ist einfacher zu zeigen und schneller verständlich.
